# Youtube-Trending-Analysis

## About
This repository contains data cleaning and exploratory data analysis (EDA) of the YouTube Trending Dataset (Kaggle) using Python. The analysis is focused on trending videos data from regions such as the US.

## Project Goals
- Clean and preprocess YouTube trending video datasets.
- Perform EDA to uncover insights and trends in video popularity.
  
## Technologies Used
- Python
- Jupyter Notebook
- Pandas, Seaborn, Matplotlib

## Repository Structure
- **Datasets**: Contains cleaned and raw datasets (if included).
- **Notebooks**: Jupyter Notebooks with code for data cleaning, EDA.
- **README.md**: This documentation file.
